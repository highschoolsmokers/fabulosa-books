# Image Placement Guide for the Tutorial

Where each image should go in the Substack post. For each entry: the image file, where it sits in the text, and a suggested caption.

---

## Part 1: Welcome to Fabulosa Books

### 1. Photo of Fabulosa Books
- **File:** `fabulosa-storefront.jpg` (or whatever you name your photo)
- **Placement:** Right after the title, before "A Bookstore with Heart"
- **Caption:** *Fabulosa Books, San Francisco's Castro neighborhood*

## Tips for Substack

- Substack supports JPG, PNG, GIF, and SVG uploads
- SVGs may render differently across email clients — consider converting to PNG for email-heavy audiences
- Recommended image width: 700px (matches Substack content area)
- Keep file sizes reasonable — compress PNGs if needed
- Alt text matters for accessibility — use the captions as alt text
